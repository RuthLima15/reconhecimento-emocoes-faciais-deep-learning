# Reconhecimento de Emoções Faciais com Deep Learning

## Objetivo

Desenvolver um sistema capaz de reconhecer emoções faciais utilizando técnicas de Deep Learning.

## Dataset

FER2013

## Tecnologias

- Python
- TensorFlow
- Keras
- Streamlit
- SQLite
- NumPy
- Pandas
- Scikit-Learn

## Modelos treinados

- DNN
- CNN
- MobileNetV2
- ResNet50

## Métricas

- Accuracy
- Precision
- Recall
- F1-score
- Curva ROC
- Matriz de Confusão

## MVP

Aplicação desenvolvida em Streamlit permitindo:

- Upload da imagem
- Predição da emoção
- Histórico das predições
- Armazenamento das interações em SQLite
